# 🚀 Realtime Chat App

A real-time chat application built with Node.js, Express, and Socket.io.

## 🌟 Features

- [x] Real-time messaging via WebSockets\n- [x] Multiple chat rooms support\n- [x] User presence indicators

## 🛠️ Tech Stack

- **Node.js**\n- **Express**\n- **Socket.io**\n- **Vanilla JS**

---

## 🚀 How to Download & Run

Follow these steps to set up the project on your local machine.

1. Clone the repository:
   ```bash
   git clone https://github.com/Aliffhafizirosdi/realtime-chat.git
   ```
2. Navigate to the directory:
   ```bash
   cd realtime-chat
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000`
